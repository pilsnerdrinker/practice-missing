# ミッシング 1238/4567 通し練習ツール

FINAL FANTASY XIV の高難易度コンテンツに登場する「ミッシング」処理を、  
1回目から8回目まで通しで練習するための非公式ファンメイド練習ツールです。

## 概要

このツールでは、初期マークを見て、自分のロールが各回でどこに立つか・何を処理するかを練習できます。

対応している主な要素は以下です。

- KT（りーん）式 / ヤーン式の切り替え
- ノーマル / ハードの切り替え
- 先組・後組による 1238 / 4567 処理
- 塔踏み位置の判定
  - 左塔・上
  - 右塔・上
  - 左塔・下
  - 右塔・下
- 頭割り位置の判定
- 扇誘導位置の判定
- 過去/未来の終焉（消滅の脚（半面焼き）の向き決定）
- 消滅の脚（半面焼き）の誘導
- 塔を踏んだ後のマーク変化表示

## 使い方

1. 解法を選びます。
   - `KT（りーん）`
   - `ヤーン式`

2. 難易度を選びます。
   - `ノーマル`
   - `ハード`

3. 自分のロールを選びます。

4. 表示された初期マークを確認します。

5. 各回の正しい行動を選択します。

6. 判定ボタンで正誤を確認します。

## ノーマルモード

ノーマルモードでは、各回の行動をボタンで選択します。

主に以下のような判断を練習します。

- どちらの塔を踏むか
- 塔の上側/下側のどちらに入るか
- 左右どちらの頭割りに入るか
- 左右どちらの扇を誘導するか
- ボス左上/右上で過去未来の円範囲を誘導するか
- 消滅の脚でボスを塔側/塔の逆側のどちらへ向けるか

## ハードモード

ハードモードでは、図の中から正しい立ち位置をクリックして選びます。

流れは以下のように、実際の処理順に近い形で進みます。

```text
1回目
→ 2回目
→ 消滅の脚誘導
→ 3回目
→ 4回目
→ 消滅の脚誘導
→ 5回目
→ 6回目
→ 消滅の脚誘導
→ 7回目
→ 8回目
→ 最後の消滅の脚誘導
```


ハードモードでは、以下のような位置を図上で選択します。

```text
左塔・上
右塔・上
左塔・下
右塔・下
左頭割り
右頭割り
左上扇誘導
左下扇誘導
右上扇誘導
ボス左上
ボス右上
脚：塔側へ
脚：塔の逆へ
```

## 解法について

初期付与は、TH組に頭割り1人、DPS組に頭割り1人が付きます。

### KT（りーん）式

KT（りーん）式では、以下のペア分けを想定しています。

```text
MT-ST
H1-H2
D1-D2
D3-D4
```

頭割り2人とペア内優先順位を基準に先組/後組を決定し、1/2/3/8 または 4/5/6/7 を処理します。

### ヤーン式

ヤーン式では、以下のペア分けを想定しています。

```text
MT-H1
ST-H2
D1-D3
D2-D4
```

頭割りがついた人を含むペアが先組になります。

優先順位は以下を想定しています。

H1 > H2 > MT > ST > D1 > D2 > D3 > D4

## マーク変化について

塔を踏むと、塔を踏んだ4人だけが次回用のマークに変化します。

奇数回に塔を踏んだ場合：

円2 / 扇2

偶数回に塔を踏んだ場合：

頭割り2 / 円1 / 扇1

このツールでは、各回ごとに

踏む前のマーク
踏んだ後のマーク（次回用）

を表示します。
