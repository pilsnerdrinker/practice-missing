# ミッシング 1238/4567 通し練習ツール

P2ミッシング1回目から8回目まで通しで練習するための非公式ファンメイド練習ツールです。

## 概要

このツールでは、初期マークを見て、自分のロールが各回でどこに立つか・何を処理するかを練習できます。

対応している主な要素は以下です。

- KT（りーん）式 / ヤーン式・優先順 / ヤーン式・南調整の切り替え
- ギミック理解 / 位置確認 / 実戦練習の切り替え
- 先組・後組による 1238 / 4567 処理
- 塔踏み位置の判定
  - 左塔・上
  - 右塔・上
  - 左塔・下
  - 右塔・下
- 頭割り位置の判定
- 扇誘導位置の判定
- 過去/未来の終焉（消滅の脚（半面焼き）の向き決定）
- 消滅の脚（半面焼き）の誘導
- 塔を踏んだ後のマーク変化表示

## 使い方

1. 解法を選びます。
   - `KT（りーん）式`
   - `ヤーン式・優先順`
   - `ヤーン式・南調整`

2. モードを選びます。
   - `ギミック理解`
   - `位置確認`
   - `実戦練習`

3. 自分のロールを選びます。

4. 表示された初期マークを確認します。

5. 各回の正しい行動を選択します。

6. 判定ボタンで正誤を確認します。

実戦練習モードでは、スタート後に3カウントが表示され、一定時間内にマップ上の立ち位置を選択します。

## ギミック理解モード

ギミック理解モードでは、各回の行動をボタンで選択します。

主に以下のような判断を練習します。

- どちらの塔を踏むか
- 塔の上側/下側のどちらに入るか
- 左右どちらの頭割りに入るか
- 左右どちらの扇を誘導するか
- ボス左上/右上で過去未来の円範囲を誘導するか
- 消滅の脚でボスを塔側/塔の逆側のどちらへ向けるか

## 位置確認モード

位置確認モードでは、図の中から正しい立ち位置をクリックして選びます。

流れは以下のように、実際の処理順に近い形で進みます。

```text
1回目
→ 2回目
→ 消滅の脚誘導
→ 3回目
→ 4回目
→ 消滅の脚誘導
→ 5回目
→ 6回目
→ 消滅の脚誘導
→ 7回目
→ 8回目
→ 最後の消滅の脚誘導
```


位置確認モードでは、以下のような位置を図上で選択します。

```text
左塔・上
右塔・上
左塔・下
右塔・下
左頭割り
右頭割り
左上扇誘導
左下扇誘導
右上扇誘導
ボス左上
ボス右上
脚：塔側へ
脚：塔の逆へ
```

## 実戦練習モード

実戦練習モードでは、処理順に合わせてマップ上の立ち位置を選択します。

主な流れは以下です。

```text
スタート
→ 3カウント
→ 各処理の詠唱/確認時間
→ 回答
→ 正解/不正解表示
→ 次の処理へ
```

実戦練習モードでは、以下の要素を練習できます。

- 3カウント後に処理開始
- 処理ごとの制限時間表示
- 残り時間バー表示
- マップ上の立ち位置選択
- 正解/不正解の即時表示
- 1問ごと / 連続モードの切り替え
- 1問前に戻る機能
- 塔の位置を固定するモード
- 通しログでの結果確認

連続モードでは、最後まで通したあとにログが表示され、一定時間後に次の問題へ進みます。

「塔の位置を固定」にチェックを入れると、マップが回転しない状態で練習できます。  
塔位置を覚える練習をしたい場合はチェックなし、処理手順だけを確認したい場合はチェックありで使えます。

正解表示では、自分の正解位置を太い黄色丸、見る相手を細い黄色丸で表示します。  
次の処理で見るマークも、正解表示中から確認できます。


## 解法について

初期付与は、TH組に頭割り1人、DPS組に頭割り1人が付きます。

### KT（りーん）式

KT（りーん）式では、以下のペア分けを想定しています。

```text
MT-ST
H1-H2
D1-D2
D3-D4
```

頭割り2人とペア内優先順位を基準に先組/後組を決定し、1/2/3/8 または 4/5/6/7 を処理します。

### ヤーン式・優先順

ヤーン式・優先順では、以下のペア分けを想定しています。

```text
MT-H1
ST-H2
D1-D3
D2-D4
```

頭割りがついた人を含むペアが先組になります。

左右を決める優先順位は以下を想定しています。

```text
H1 > H2 > MT > ST > D1 > D2 > D3 > D4
```

### ヤーン式・南調整

ヤーン式・南調整では、ペア分けと先組/後組の決め方はヤーン式・優先順と同じです。

```text
MT-H1
ST-H2
D1-D3
D2-D4
```

基本は、同じ塔を踏んだ2人のマーカーを見て、次回の塔位置を調整します。

- マーカーがかぶっていない場合：同じ塔を踏みます。
- マーカーがかぶっている場合：南側を踏んだ人が逆の塔へ移動します。

このツールでは、ヤーン式・南調整のときだけ、同じ塔を踏む2人をマーク表示欄で破線表示します。

- 水色の破線：自分
- グレーの破線：同じ塔を踏む相手

4回目は、必ずTH側・DPS側それぞれでマーカーがかぶるため、以下の固定配置を想定しています。

```text
H / 近接：左塔
T / 遠隔：右塔
各塔で 扇＝上、円＝下
```

3回目の塔処理後、自分についたマーカーによって8回目用のマーカーを付与する想定です。

```text
円：禁止1 / 禁止2
扇：バインド1 / バインド2
```

8回目の塔処理は、以下の数字マーカー基準で判定します。

```text
バインド1：左塔・上
禁止1：左塔・下
バインド2：右塔・上
禁止2：右塔・下
```

## マーク変化について

塔を踏むと、塔を踏んだ4人だけが次回用のマークに変化します。

奇数回に塔を踏んだ場合：

円2 / 扇2

偶数回に塔を踏んだ場合：

頭割り2 / 円1 / 扇1

このツールでは、各回ごとに

踏む前のマーク
踏んだ後のマーク（次回用）

を表示します。

ヤーン式・南調整では、同じ塔を踏む2人を破線で表示します。  
また、3回目後には8回目用の禁止/バインドマーカーも表示します。

実戦練習モードでは、回答後に次の処理で見るマークも表示します。
